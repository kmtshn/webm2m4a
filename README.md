🎧 webm2m4a（社内向けツール）

Webブラウザ上で .webm ファイルを .m4a に変換できるツールです。
ソフトウェアのインストール不要で、安全に音声ファイルを変換できます。

📌 概要

本ツールは、Web会議や録音データなどで利用される .webm 形式の音声ファイルを、
汎用性の高い .m4a 形式へ変換することを目的としています。

変換処理はすべてブラウザ内で実行されるため、
ファイルが外部サーバーへ送信されることはありません。

🚀 主な特徴

🌐 ブラウザのみで動作（インストール不要）

🔒 ファイルはローカル処理（外部送信なし）

⚡ シンプル操作で誰でも利用可能

📦 HTML単体で動作（配布が容易）

🧭 利用シーン

Web会議の録音データ（WebM）をスマートフォンで再生したい場合

音声ファイルを社内標準フォーマット（M4A）に統一したい場合

セキュリティ上、外部サービスを利用せず変換したい場合

🖥️ 利用方法

本ツールをブラウザで開く

.webm ファイルを選択

「変換」ボタンをクリック

.m4a ファイルをダウンロード

※ドラッグ＆ドロップ対応の場合はその旨追記してください

🔐 セキュリティについて

本ツールは以下の特徴により、安全に利用できます。

ファイルはローカル環境でのみ処理

外部サーバーへのアップロードは行いません

インターネット接続がなくても利用可能（ローカル保存時）

🛠️ 技術概要

HTML / JavaScript

WebAssembly（FFmpeg等）

Web Audio API

⚠️ 注意事項

大容量ファイルは処理に時間がかかる場合があります

ブラウザの性能に依存します（推奨：Chrome / Edge 最新版）

一部の古いブラウザでは動作しない可能性があります
